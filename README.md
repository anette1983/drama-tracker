# EastAsian Serenity Tracker

A Next.js app for discovering and tracking Korean and Chinese dramas/movies.

## Project Docs

- Product blueprint: [docs/blueprint.md](docs/blueprint.md)
- Backend data model: [docs/backend.json](docs/backend.json)
- Delivery roadmap and prioritized backlog: [docs/roadmap.md](docs/roadmap.md)

## Current Focus

This project is being built with a free-tier-first constraint:

- Firebase Spark plan for auth and data.
- TMDb API for content discovery.
- Rule-based recommendations as the default production path.
- Optional AI recommendations behind a feature flag.
